# ミッション09: MCP でツールを接続する

## ゴール

- MCP サーバーを追加し、Claude Code が「ファイルとコマンド以外の世界」に触れる体験をする
- 題材アプリの SQLite DB を自然言語で調べる

## 背景解説

### MCP とは

MCP (Model Context Protocol) は、AI に外部ツールを接続するための共通規格です。Claude Code は素の状態でもファイル操作とコマンド実行ができますが、MCP サーバーを足すと**ツールそのもの**が増えます。

- DB サーバー → SQL を組み立てて直接クエリできる
- ブラウザ MCP → 実際にページを開いて動作確認できる
- Backlog/GitHub MCP → 課題やPRを読み書きできる

「コマンド経由でもできること」との違いは確実性です。例えば sqlite3 コマンドが未インストールの環境でも、MCP サーバーがあれば DB を扱えますし、ツールの入出力が構造化されているため誤操作が減ります。

### 設定の仕組み

```bash
claude mcp add <名前> -- <起動コマンド>   # 追加
claude mcp list                           # 一覧
claude mcp remove <名前>                  # 削除
```

設定はスコープ（このプロジェクトだけ/ユーザー全体）を選べます。**追加後は新しいセッションを開くまでツールが認識されない**点に注意。

## ミッション手順

### 1. DB にデータを用意する

```bash
cd app
npm run seed
```

`data/kintai.db` に8件のサンプルデータが入ります。

### 2. SQLite MCP サーバーを追加する

```bash
claude mcp add kintai-db -- npx -y mcp-server-sqlite-npx "$(pwd)/data/kintai.db"
```

> MCP サーバーのパッケージは日々増えています。上記が動かない場合は「SQLite MCP サーバーの現在の定番を調べて設定して」と Claude 自身に頼むのも実践的な解決方法です。

### 3. 新しいセッションで DB を探索する

`claude` を起動し直して:

```
kintai-db にどんなテーブルがある？スキーマを見せて
```

```
2026年7月の勤怠で、退勤が19時以降の日を教えて
```

```
打刻忘れ（clock_out が NULL）のレコードは何件？
```

SQL を自分で書かずに DB の中身が調べられることを確認してください。裏で Claude がどんな SQL を組み立てたかも表示されるので、**SQL の学習教材としても**使えます。

### 4. アプリのデバッグに使う

ミッション03の発展課題（APIの500エラー）をやった人は、その原因調査を MCP 込みでやり直してみてください:

```
/api/summary/2026-07 が500になる。kintai-db の実データを確認しながら原因を調査して
```

コード読解 + 実データ確認を組み合わせた調査は、実務のデバッグそのものです。

### 良い頼み方 / 悪い頼み方

> ❌ （MCP を追加しただけで満足して）「DBをいい感じに分析して」
>
> ✅ 「打刻忘れの件数と、該当する日付・備考を表で出して」

ツールが増えても3原則は変わりません。MCP は Claude の手足を増やすだけで、頭への指示はあなたの仕事です。

## 発展課題

- チームで使っている外部サービス（Backlog、Redmine、Slack等）の MCP サーバーがあるか調べる
- MCP サーバーの実体を理解する: 「MCP サーバーって中身は何をしてるの？stdin/stdout で何が流れてる？」と Claude に聞いて仕組みを掘る

## ハマりどころ

| 症状 | 対処 |
|---|---|
| 追加したのにツールが見えない | **新しいセッションが必要**。既存の会話には現れない |
| npx のダウンロードが遅い/失敗 | ネットワーク要因が多い。リトライ、またはパッケージを事前 install |
| DB ファイルが見つからないエラー | 相対パスは起動ディレクトリ基準でずれやすい。上記のように `$(pwd)` を使った絶対パス指定が安全 |
| どの MCP サーバーを選べばいいか分からない | 公式・メンテが活発なものを優先。Claude に「候補を比較して」と調べさせるのも手 |
