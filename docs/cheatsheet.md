# Claude Code チートシート

## 起動・セッション

| 操作 | 説明 |
|---|---|
| `claude` | 対話セッション開始 |
| `claude "質問"` | ワンショット実行 |
| `claude -c` | 直前のセッションを再開 |
| `/clear` | 会話をリセット（タスクごとに実行が基本） |
| `/compact` | 会話を要約して圧縮（同タスク継続時） |
| `Esc` | 生成を中断 |
| `Shift+Tab` | 権限モード切替（通常/auto-accept/plan） |

## よく使うスラッシュコマンド

| コマンド | 説明 |
|---|---|
| `/help` | ヘルプ |
| `/init` | CLAUDE.md の雛形生成 |
| `/config` | 設定画面 |
| `/mcp` | MCP サーバーの状態確認 |
| `/自作skill名` | 自作 skill の呼び出し |

## 設定ファイルの置き場所

| パス | スコープ | 用途 |
|---|---|---|
| `~/.claude/CLAUDE.md` | ユーザー全体 | 個人の好み（応答言語など） |
| `<repo>/CLAUDE.md` | プロジェクト | プロジェクトの前提・規約（チーム共有） |
| `<repo>/.claude/skills/<name>/SKILL.md` | プロジェクト | 手順書（チーム共有可） |
| `<repo>/.claude/settings.json` | プロジェクト | hooks・権限（チーム共有） |
| `~/.claude/settings.json` | ユーザー全体 | 個人の hooks・権限 |

## MCP

```bash
claude mcp add <名前> -- <起動コマンド>   # 追加(新セッションから有効)
claude mcp list                           # 一覧
claude mcp remove <名前>                  # 削除
```

## 頼み方の3原則（ミッション01・03）

1. **コンテキストを渡す** — 何のために・どこを対象に
2. **タスクを分割する** — 確認可能な単位に
3. **曖昧さを排除する** — 正解の基準・範囲・観点を指定

## 検証の習慣（ミッション02）

```bash
git diff     # AIの変更は必ず自分の目で読む
npm test     # 自分でも実行する
```

- 「テスト通るようにして」ではなく「実装のバグを直して。テストは変更しない」
- 変更を自分の言葉で説明できないなら、受け入れない

## 使い分け早見表

| 実現したいこと | 使うもの |
|---|---|
| 毎回守ってほしい前提・規約 | CLAUDE.md |
| 繰り返し使う手順書 | skill |
| 絶対に強制したい処理・禁止 | hooks |
| 外部ツール・データへの接続 | MCP |
| 大量に読む調査の委譲 | サブエージェント |
| 複数タスクの完全並列 | git worktree + 複数セッション |
| チーム全体への展開 | GitHub Actions 連携 |
